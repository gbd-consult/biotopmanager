Installation
============

Das QGIS Plugin ``GBD Biotopmanager`` wird über das Plugin Repository der Geoinformatikbüro Dassau GmbH bereitgestellt. Sie können das Repository über den QGIS Pluginmanager einbinden.

.. figure:: ./screenshots/repodetails.png
  :align: center
  :width: 400

Das Plugin kann über das ``Menü Erweiterungen`` -> ``GBD Biotopmanager`` oder über die Werkzeugleiste ``GBD Biotopkataster`` geladen werden.

Wenn Sie das Plugin ``GBD Biotopmanager`` in QGIS geladen und geöffnet haben, finden Sie folgende Werkzeugleiste vor:

.. figure:: ./screenshots/werkzeugleiste.png
  :align: center

Mit gedrückter, linker Maustaste auf das geriffelte, linke Ende der Werkzeugleiste können Sie die Icons an einen beliebigen Ort verschieben, wie andere Werkzeugleisten in QGIS auch.
